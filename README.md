# אחוזת המשכן — אתר וילה ביבנאל

אתר תדמית לוילה יוקרתית להשכרה ביבנאל, גליל תחתון.
אתר חי: [shablonishay.github.io/HaMishkan](https://shablonishay.github.io/HaMishkan/)

---

## מבנה הקבצים

```
index.html      עמוד ראשי (כל התוכן בקובץ אחד)
style.css       כל העיצוב (RTL, רספונסיבי, אנימציות)
script.js       גלריה אוטומטית, אנימציות גלילה
favicon.png     אייקון הלשונית בדפדפן
assets/
  hero.jpeg     תמונת הרקע של הדף הראשי
  pool.jpeg     תמונת הבריכה בסקציה הייעודית
  LOGO.png      לוגו האחוזה
  2.jpg–22.jpg  תמונות הגלריה (ממוספרות)
```

## הוספת תמונות לגלריה

הסקריפט טוען אוטומטית קבצים בשם `N.jpg` / `N.jpeg` (N = 2 עד 99) מתיקיית `assets/`.

להוספת תמונה: פשוט שמור קובץ בשם `23.jpg`, `24.jpg` וכו' בתיקיית `assets/` — הגלריה תקלוט אותו אוטומטית.

## SEO

תגיות שמוגדרות ב-`index.html`:

- `<meta name="description">` — תיאור לתוצאות גוגל
- Open Graph — שיתוף ב-WhatsApp / Facebook
- Twitter Card — שיתוף ב-Twitter / X
- Geo tags — מיקום יבנאל לחיפוש מקומי
- Schema.org `LodgingBusiness` (JSON-LD) — מידע עשיר בגוגל

**בדיקת Schema:** [Google Rich Results Test](https://search.google.com/test/rich-results)
